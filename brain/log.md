# Log — what happened, in order

> **Workshop TL;DR (for humans):** A running diary of everything the brain has done — every compile,
> query, and check, newest at the bottom. Each line starts the same way so it's easy to scan. You'll add
> a line here in most exercises.

---

Format: `## [YYYY-MM-DD] <operation> | <subject>` where operation is `ingest` (compile), `query`, or
`lint`.
